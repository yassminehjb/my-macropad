# my-macropad
My Hackpad Project
Welcome to my 3-key macropad project built for the Hack Club Hackpad challenge.

This device is designed to map three mechanical switches to custom shortcuts or keys.

Project Structure
Here is what is inside this repository:

CAD: The 3D model of the enclosure (STEP/3MF files) designed in Fusion 360.

PCB: The electronic schematic and board design files created with KiCad.

Firmware: The CircuitPython/KMK code (main.py) that powers the keys.

production: The final manufacturing files, including the Gerber files (gerbers.zip) for the PCB factory and the .stl file for the 3D printer.

Hardware Components
To build this project, you will need:

1x Seeed Studio XIAO RP2040

3x Mechanical Switches

3x Keycaps

1x Custom PCB

1x Custom 3D Printed Case

Software and Firmware
The macropad runs on KMK Firmware using CircuitPython. It maps the 3 keys to basic shortcuts or keys (like A, B, and C) to make sure everything works perfectly.
